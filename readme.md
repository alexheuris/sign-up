# Sign up

## Dependencies

- Dotnet Core SDK
- Docker
- Docker Compose

## Restore

From the project root, run:
```
dotnet restore
```

## Building

From the project root, run:
```
dotnet build
```

## Publish

From the project root, run:
```
./publish.sh
```

## Running

From the project root, run:
```
docker-compose up
```

## Stopping

From the project root, run:
```
Ctrl+C
docker-compose down
```
